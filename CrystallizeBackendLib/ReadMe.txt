Crystallize Backend Library
-----------------------------------------
-----------------------------------------
The Crystallize Backend Library contains the code to facilitate communication between the Crystallize game code and the JAVA API which is responsible to connect to DynamoDB, a NoSQL Database Amazon Web Services.
The code makes HTTP/HTTPS web requests to the JAVA based server hosted on Amazon cloud to perform specific operations like
	a) Query data from the database
	b) Save data to the database
	c) Delete data from the database
	d) Create tables in database
	e) Delete tables from databse
